# Lesson-25.Rsyslog
# Домашнее задание 25: Vagrant c Rsyslog

## Задания
Поднимаем две машины — web и log.<br>
На web поднимаем nginx.<br>
Настраиваем центральный лог-сервер на любой системе по выбору:<br>
journald;<br>
rsyslog;<br>
elk.<br>
Настраиваем аудит, который будет отслеживать изменения конфигураций nginx.<br>
Все критичные логи с web должны собираться и локально и удаленно.<br>
Все логи с nginx должны уходить на удаленный сервер (локально только критичные).<br>
Логи аудита должны также уходить на удаленную систему.<br>

Формат сдачи ДЗ - vagrant + ansible
